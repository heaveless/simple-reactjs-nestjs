# React - NestJs ԅ(≖‿≖ԅ)

**Install**

```
git clone https://github.com/heaveless/simple-reactjs-nestjs.git
```

**backend**

- NestJs
- Node 18.18.0

*Install dependencies*
```
npm i
```

*run project*
```
npm run start:dev
```

*location*
```
http://localhost:3000
```

*test requests*
```
GET /books
POST /books/create

GET /books
GET /books/pages-per-chapter/:id
POST /books/create
```

**frontend**

- React
- Node 18.18.0

*Install dependencies*
```
npm i
```

*run project*
```
npm run dev
```

*location*
```
http://localhost:5000
```
